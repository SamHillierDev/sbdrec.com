SBDREC is a powerlifting record keeper.

## Requirements

Make sure you have Node and NPM installed:

```bash
node -v
# v20.10.0
```

```bash
npm -v
# 10.9.0
```

## Installation

Run the commands:

```bash
npm install
```

```bash
npm run dev
```
